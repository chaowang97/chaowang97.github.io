搭建hexo博客

2019-07-06 16:00:40

第一步

下载node.js

第二步

管理员运行cmd

第三步

输入node -v 和npm -v 查看版本

第四步

更换远程仓库

npm install -g cnpm –registry=https://registry.npm.taobao.org

第五步

下载hexo组件

cnpm install -g hexo-cli

第六步

查看hexo -v版本

hexo -v

第七步

创建博客目录，我选择到D盘去创建
d:

mkdir hexoblog 创建一个hexoblog文件夹

第八步

创建hexo博客

hexo init

第九步

启动hexo

hexo s

第十步

创建博客文章

hexo n “我的第一篇博客”

第十一步

清理hexo

hexo clean

第十二步

生成博客

hexo g

第十三步

再次启动hexo

hexo s
